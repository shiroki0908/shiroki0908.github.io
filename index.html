从零开始butterfly博客的魔改教程(一)
前言这是一篇从零开始构建butterfly博客的教程，主要面向的完全不知道如何构建Hexo博客的小白，这篇教程会从构建hexo博客开始，初步目标是给Hexo安装上本次的主人公butterfly主题。 Hexo的依赖要求 Node.js (Node.js 版本需不低于 .，建议使用 Node.js . 及以上版本) Git . 安装Git这里我们将介绍不同操作系统的安装方式。 Windows: 去指定官网Git官网下载安装包安装。 Linux: 一般Linux系统都自带Git，如果没有，可以用包管理器安装。 Ubuntu;Debian: sudo apt-get install git CentOS;Red Hat;Fedora: sudo yum install git Arch Linux: sudo pacman -S git macOS: 去指定官网Git官网下载安装包安装。 . 安装Node.js这里我们将介绍不同操作系统的安装方式。 Windows: 去指定官网Node.js官网下载安装包安装。 Linux: 一般Linux系统都自带Node.js，如果没有，可以用包管理器安装。 Ubuntu;Debian: sudo apt-get install nodejs CentOS;Red Hat;Fedora: sudo yum install nodejs Arch Linux: sudo pacman -S nodejs macOS: 去指定官网Node.js官网下载安装包安装。 安装 Hexo所有必备的应用程序安装完成后，即可使用 npm 安装 Hexo。 . 安装Hexo客户端 Windows: 在命令提示符中运行 npm install -g hexo-cli Linux: 在终端中运行 sudo npm install -g hexo-cli macOS： 在终端中运行 sudo -s npm install -g hexo-cli . 执行Hexo命令 Linux以外执行npx hexo &lt;command&gt; Linux用户可以将 Hexo 所在的目录下的 node_modules 添加到环境变量之中即可直接使用npx hexo &lt;command&gt;echo &#;PATH=&quot;$PATH:./node_modules/.bin&quot;&#; &gt;&gt; ~/.profile . 安装 Hexo 完成后，需要再指定文件夹内新增文件，执行hexo init命令，Hexo会在指定文件夹内生成所需文件。 $ hexo init &lt;folder&gt; cd &lt;folder&gt; npm install macOS用户安装需要开启管理员，所以需要在终端中运行sudo npm install 初始化后，您的项目文件夹将如下所示：.├── _config.yml├── package.json├── scaffolds├── source| ├── _drafts| └── _posts└── themes . 执行Hexo三板斧 hexo cl在终端中运行hexo cl命令，确认刺青。(hexo cl是hexo clean的简写) hexo g在终端中运行hexo g命令，生成静态文件。(hexo g是hexo generate的简写) hexo s在终端中运行hexo s命令，启动本地服务器。(hexo s是hexo server的简写) 在本地端口确认Hexo博客是否正常运行。 若成功运行则会显示成下图这样: 接下来，我们可以开始安装butterfly主题了。 安装butterfly主题. 通过Git获取源代码Stable branch [recommend]:git clone -b master https://github.com/jerryc/hexo-theme-butterfly.git themes/butterfly Dev branch:git clone -b dev https://github.com/jerryc/hexo-theme-butterfly.git themes/butterfly . NPM安装Butterfly主题在终端中运行npm install hexo-theme-butterfly --save命令，安装Butterfly主题。（macOS用户需要开启管理员，所以需要在终端中运行sudo npm install hexo-theme-butterfly --save） . 修改Hexo配置文件打开Hexo配置文件_config.yml，找到theme项，将theme的值修改为butterfly。 如果你没有pug和stylus renderer，请在终端中运行以下代码:npm install hexo-renderer-pug hexo-renderer-stylus --save命令安装。(macOS用户需要开启管理员，所以需要在终端中运行sudo npm install hexo-renderer-pug hexo-renderer-stylus --save) . 启用Butterfly主题执行hexo三板斧命令，确认无误后，在浏览器中访问本地端口，即可看到Butterfly主题的博客页面。 如果成功安装，你将看到如下画面: 总结以上就是butterfly主题的傻瓜式安装教程，如果你遇到了问题，欢迎在评论区留言，希望能帮助到大家。
