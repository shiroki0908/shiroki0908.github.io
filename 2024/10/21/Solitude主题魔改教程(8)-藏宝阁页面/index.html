Solitude主题魔改教程(8)-藏宝阁页面

文章摘要

前言

Solitude主题默认没有藏宝阁页面功能，但有时我们希望展示个人收藏的番剧、小说、电影、漫画等内容。本文将详细介绍如何从blog-solitude项目完整移植藏宝阁页面功能到solitude-latest，包括页面配置、模板创建、数据管理和样式设计的完整实现方案。

现象与需求

现象

  • Solitude主题默认没有藏宝阁页面
  • 缺少收藏展示和分类功能
  • 无法展示个人收藏的番剧、小说、电影等
  • 缺少收藏数据管理机制

期望效果

  • 独立的藏宝阁页面（/collects/）
  • 支持收藏分类展示（番剧、小说、电影、漫画等）
  • 每个收藏包含封面、名称、评分、标签、描述
  • 支持豆瓣链接跳转
  • 响应式设计，适配不同屏幕尺寸
  • 视频Banner支持
  • 悬停效果和动画

问题分析

根本原因

  1. 页面类型缺失
    solitude-latest主题缺少collects页面类型的支持

  2. 模板文件缺失
    缺少collects页面的模板文件和内容组件

  3. 数据管理缺失
    缺少收藏数据的配置和管理机制

  4. 样式文件缺失
    缺少collects页面的CSS样式文件

  5. 路由配置缺失
    页面路由中缺少collects类型的处理

修改文件清单

  • solitude-latest/source/collects/index.md
  • solitude-latest/source/_data/collects.yml
  • solitude-latest/node_modules/hexo-theme-solitude/layout/page.pug
  • solitude-latest/node_modules/hexo-theme-solitude/layout/includes/page/collects.pug
  • solitude-latest/node_modules/hexo-theme-solitude/source/css/_page/collects.styl
  • solitude-latest/node_modules/hexo-theme-solitude/source/css/index.styl

数据结构说明：

  • class_name - 分类名称（支持emoji）
  • item_list - 该分类下的收藏列表
  • name - 收藏名称
  • image - 封面图片链接
  • rating - 评分（0-10分）
  • icon - Font Awesome图标类名
  • type - 收藏类型
  • date - 创建日期
  • tags - 标签数组
  • tag_colors - 标签颜色数组
  • description - 详细描述
  • douban_link - 豆瓣链接

  • 日期格式建议使用 YYYY/MM/DD 或 YYYY-MM-DD

总结

通过以上步骤，我们成功为Solitude主题添加了完整的藏宝阁功能。这个功能不仅提供了美观的收藏展示界面，还具有良好的扩展性和维护性。用户可以根据自己的需求轻松添加和管理收藏内容，展示个人的兴趣爱好和品味。
